Project six
Falafel project
A Falafel Restaurant Ordering System project aimed at simplifying and organizing the ordering process in small restaurants.
The idea of the application is to help staff manage orders in a faster and more organized way. Customer orders are registered in the system, and staff receive notifications that include the order details and the total amount, which helps reduce errors and speed up the service.
The system supports a menu that includes several items such as falafel, hummus, foul, kebda, and musabaha, allowing orders to be recorded and processed easily.
The application also includes a timing system for managing orders, which helps small restaurants organize the preparation time and track how long each order takes, improving service speed and workflow inside the restaurant.
